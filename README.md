# 1.프로젝트 배경 및 주제
1) 프로젝트 배경  
서울의 지하철 이용객 수가 늘어나고 문화 시설 이용률도 늘어남에 따라  
대중 교통과 문화 시설을 연관 지어서 여러 사람들이 다양한 데이터들의 접근이 편하면 좋겠다고 생각해서   
지하철 역 주변의 문화시설의 정보를 제공하는 대시보드를 기획하게 되었다.
